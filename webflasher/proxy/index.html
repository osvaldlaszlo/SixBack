SixBack OAuth Proxy

Initialisiere…

Diese Seite leitet einen OAuth-Callback (von Spotify, ggf. andere Provider) zurück an deine lokale SixBack-Instanz im LAN. Die Seite ist rein statisch — kein Server-Code, kein Logging. Code + state werden nur im Browser ausgewertet und per Top-Level-Navigation an das LAN-Ziel weitergereicht.
